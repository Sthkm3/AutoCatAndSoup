# **AutoCatAndSoup**

Cat and Soup auto script | 貓咪與湯輔助腳本，基於Autojs6撰寫和編譯。

![icon](https://play-lh.googleusercontent.com/A2Wq3C8LfjxpREC0DccMcYcRIqWB7ol5o3cDhfL3rYS2ZFqOwCB5tFOEOS5Tzya37df7=s180-rw)

## 功能

### 基本任務

- **販賣商品**：有商品出現在販賣區時，自動點擊以收取金幣。
- **設施升級**：在金幣足夠時，升級貓咪們的設施。
- **食譜升級**：在體力足夠時，升級貓咪們的食譜 (後期食譜滿等可關閉)。

### 獎勵領取

- **掛機獎勵**：領取離線獎勵 (通常極限為離線六個小時)。
- **罐子精靈獎勵**：領取隨機罐子精靈獎勵。
- **商人獎勵**：每天領取兩次商人獎勵。
- **黃金蝴蝶獎勵**：黃金蝴蝶出現時，定位到該位置後捕捉。
- **青蛙王子獎勵**：青蛙王子出現時，定位到該位置後領取。
- **信箱**：領取信箱獎勵。
- **成就**：領取成就與每日任務獎勵。

> **為何有時無法收取黃金蝴蝶和青蛙王子獎勵？** 有時黃金蝴蝶和青蛙王子出現在太靠近邊緣的地方，導致點擊時被遊戲強制拉回視角，目前腳本只依賴顏色判斷，視覺算法對手機來說太繁重而沒有使用。
> 目前可用的解決辦法是將設施移動到畫面邊緣以防止黃金蝴蝶和青蛙王子出現在太靠近邊緣的地方。

### 隨機任務

- **拍攝照片**：自動完成拍照任務並關閉結果畫面。
- **禮物**：領取貓咪們準備的禮物以及追加金幣禮物 (在非免廣告狀態下，將預設選擇中間選項)。
- **玩耍**：自動與貓咪玩耍以增加親密度。

> **會使用到廣告卷嗎？** 不會，只有在免廣告狀態下才會使用廣告加倍。

### 每日任務

> 未開發，請耐心等待後續更新。 

## 安裝

### 方法一：使用已編譯 APK（推薦）

1. **前往 Releases 頁面下載最新版本 APK**
2. **將 APK 安裝至手機**
3. **開啟應用後，依照提示啟用：**
    - 無障礙服務
    - 懸浮窗權限

### 方法二：使用 Auto.js 6 編譯

1. **安裝 Auto.js 6（AutoJS6）**
2. **將本專案下載或 clone 至手機**
3. **使用 Auto.js 6 開啟 main.js**
4. **透過 Auto.js 內建功能編譯為 APK 或直接執行**

## 使用方式

1. **開啟本應用**
2. **勾選需要啟用的功能項目**
3. **點擊「啟動」**
4. **暫時啟用螢幕截圖權限**
5. **腳本將自動啟動遊戲並開始運行**
6. **畫面左側會出現浮動視窗，可隨時：**
    - 暫停腳本
    - 繼續腳本

> 腳本運行中請勿對手機進行操作，請先暫停腳本後再操作。

> 如有執行錯誤操作可直接取消勾選該功能。 

## 環境需求

- **Android 裝置** (目前測試環境以模擬器為主，如 Bluestacks、MuMu 12、夜神模擬器等)
- **螢幕解析度：**
    - 直向 720 × 1280（目前僅支援此解析度，未來會盡力開發自由解析度）
- **遊戲版本：Cat and Soup 4.28.1**
- **需開啟：**
    - 無障礙服務
    - 懸浮窗權限
    - 截圖權限

>本專案僅供學習與研究用途，請使用者自行遵守遊戲相關服務條款，因使用本專案所產生之任何風險與後果，概由使用者自行承擔。

## 關於 **Auto.js 6**

本專案基於 Auto.js 6（AutoJS6） 開發與編譯。
Auto.js 6 是一套 Android 自動化腳本工具，透過 JavaScript 搭配無障礙服務實現點擊、滑動、畫面辨識等操作。

本專案：
- 使用 Auto.js 6 作為開發環境與編譯環境
- 最終產出為可獨立安裝的 APK
- 安裝後可在未安裝 Auto.js 的情況下直接使用
